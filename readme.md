

# Setting up venv
```
python3 --version
python3 -m venv venv
source venv/bin/activate
```


# Installing this package and getting the script files
```
pip install -e .
pip list
```


# Installing and freezing packages
```
pip freeze > requirements.txt
pip install -r requirements.txt

```




# Running scripts
```
which main3.py
main3.py
```



# Tests
## installing
```
pip install pytest
```
## running
```
pytest
```
